You can use Boot Camp Assistant on macOS to create a windows installer USB drive and then boot from it and install Windows.

Comment: This is incorrect in several ways. 1) the NVIDIA GeForce GT 330M works with Windows, but it is unsupported, meaning no official driver exists.
